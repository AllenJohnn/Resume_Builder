# Resume Builder

Professional ATS-optimized resume builder with multiple export formats.


https://buildresumeee.vercel.app/

## Features

- **Multiple Export Formats**: PDF, DOCX, HTML, TXT, JSON
- **ATS Compatibility Analyzer**: Check resume compatibility with Applicant Tracking Systems
- **Auto-save**: Automatic data persistence in browser localStorage
- **Responsive Design**: Mobile-friendly interface
- **Professional Templates**: Clean, minimalist design optimized for ATS

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- html2canvas & jsPDF
- docx
- file-saver

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view in the browser.

## Build

```bash
npm run build
```
